# Demo Video Script (60 seconds)

## 0:00–0:05 — Hook
Black screen → text fade: **"UPLINK — Live Orbital Operations"**

## 0:05–0:20 — Globe
Show mission control. Satellites moving on globe. Status bar: Kp, solar wind, comms score. Point out LIVE indicator.

## 0:20–0:35 — Pass Prediction
Search "Dubai" in observer panel. ISS pass list populates. Click a pass. Globe flies to ISS. Show azimuth/elevation/range telemetry.

## 0:35–0:50 — Space Weather
Navigate to /weather or trigger alert state. Show comms ring color change. Alert banner: "Moderate geomagnetic storm."

## 0:50–1:00 — Close
Show shareable URL: `/pass/dubai/25544`. GitHub link overlay. Tagline: **"Real orbital math. Live space weather. Mission control for Earth."**

## Recording Tips

- Record at 1920×1080, 30fps
- Use dark room / mission control aesthetic
- Mute or use subtle ambient radio static when comms > 60%
- Export as GIF for README (first 10s loop)
